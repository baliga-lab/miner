The miner-enrichment tool
=========================

This utility computes the enrichment information for a set of revised clusters generated
by miner-coexpr tool

You can see the tool's available options when you enter ``miner-enrichment -h``
at the command prompt:

.. highlight:: none

::


    usage: miner-enrichment [-h] [-mg MINGENES] [-ct CORRTHRESH]
                            expfile mapfile revclusters datadir outdir

    miner-enrichment - MINER compute enrichment

    positional arguments:
      expfile               input matrix
      mapfile               identifier mapping file
      revclusters           revised cluster JSON file
      datadir               data directory for input database
      outdir                output directory

    optional arguments:
      -h, --help            show this help message and exit
      -mg MINGENES, --mingenes MINGENES
                            min number genes
      -ct CORRTHRESH, --corrthresh CORRTHRESH
                            correlation threshold


Parameters in detail
--------------------

``miner-enrichment`` expects at least these 5 arguments:

  * **expfile:** The gene expression file a matrix in csv format.
  * **mapfile:** The gene identifier map file.
  * **revclusters:** The revised cluster JSON file generated by miner-coexpr.
  * **datadir:** The data directory that contains a sub directory "network_dictionaries" where several enrichment databases reside in pickle format.
  * **outdir:** The output directory where the results will be placed in.

In addition, you can specify the following optional arguments:

  * ``--mingenes`` or ``--mg``: the minimum number of genes.
  * ``--corrthresh`` or ``-oxt``: correlation threshold
